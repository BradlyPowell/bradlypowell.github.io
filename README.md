# Bradly's Starter CS Projects

This folder contains four beginner-friendly projects to show on applications and internships.

## 1) Portfolio Website (HTML/CSS)
Files: `index.html`, `styles.css`  
How to open: double-click `index.html`.  
How to publish free: create a GitHub repo and enable GitHub Pages (main branch, /root).

## 2) Python Quiz Game
File: `quiz.py`  
Run: `python quiz.py` (or `python3 quiz.py`)  
What it shows: logic, user input, scoring, explanations, replay loop.

## 3) Scratch Mini-Game
Open https://scratch.mit.edu/ and create a project called **"Catch the Star"** (or any title). Use:  
- Sprite 1: Player (use arrow keys to move).  
- Sprite 2: Star (glides randomly across screen).  
- Add a variable `score`.  
- Increase `score` when touching star, play sound, move star to random position.  
- Add a 30-second timer and show final score.

Add the project link to the Portfolio Website (Projects section).

## 4) Java GPA Calculator
File: `StudentGpaCalculator.java`  
Compile: `javac StudentGpaCalculator.java`  
Run: `java StudentGpaCalculator`  
What it shows: conditionals, loops, input handling, methods, weighted vs. unweighted GPA.
